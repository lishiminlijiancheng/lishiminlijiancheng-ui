# lishiminlijiancheng-ui 组件库
### 快速开始
#### 1.安装组件库
```bash
npm i lishiminlijiancheng-ui
```
#### 2.引用组件库
```javascript
//全部引入
import 'lishiminlijiancheng-ui/dist/css/index.css';
import MUI from 'lishiminlijiancheng-ui';
Vue.use(MUI);

//按需引入
import 'lishiminlijiancheng-ui/dist/css/index.css';
import { Demo } from 'lishiminlijiancheng-ui';
Vue.use(Demo)
```
