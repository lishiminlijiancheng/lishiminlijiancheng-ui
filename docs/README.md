# 快速开始
#### 安装组件库
```bash
npm i lishiminlijiancheng-ui
```
#### 引用组件库
> 在main.js中引用组件库
```javascript
//全部引入
import 'lishiminlijiancheng-ui/dist/css/index.css';
import LUI from 'lishiminlijiancheng-ui';
Vue.use(LUI);
//按需引入
import 'lishiminlijiancheng-ui/dist/css/index.css';
import {Demo} from 'lishiminlijiancheng-ui';
Vue.use(Demo);
```
